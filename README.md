> A Vue.js 2.0 UI Toolkit for Web.

## Install
```shell
npm install yun-ui-pc -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import Yun from 'yun-ui-pc'

Vue.use(Yun)

// or
import {
  Select,
  Button
  // ...
} from 'yun-ui-pc'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```

## Browser Support
Modern browsers and Internet Explorer 10+.
